powerbot is a discord [1] chat bot for the simulation, planning, and coordination of activites in “Powerplay”, a game mechanic in the video game “Elite: Dangerous” [2]. powerbot it is programmed in Ruby [3] using meew0's discordrb [4] to interact with Discord. it is built on top of an SQL database via Sequel [5], importing the community-sourced EDDB.io [6] databases. powerbot also performs timed events with rufus-scheduler [7].
